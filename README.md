# Rise and Dine

Morning munches to brighten your day!

## Info
Currently, Rise and Dine is working on Forge 1.19.3. We plan to accommodate all or most 1.19 versions on Forge and Fabric. 

## Features
 * Bagels!
   * Plain
   * Everything
   * Cream Cheese

## Planned

 * Waffles
 * Maple syrup
 * ... and more!
